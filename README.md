# News Finder using API

## Name: Siddhi Dixit

## Roll Number: 24BAI10073

## Project Description

This project is a Java-based application that retrieves real news articles based on a user’s input topic using a live API. It allows users to search for any topic and view related news from reliable sources.

---

## Problem

In today’s digital world, people often come across information from unknown or unreliable sources. It becomes difficult to verify whether the information is accurate. Instead of guessing whether news is true or false, users need access to trusted sources.

---

## Solution

This application allows users to enter a topic and fetches real news articles from reliable sources using an API. It displays headlines along with their source names so users can read and verify information themselves.

---

## Features

* Fetches real-time news based on user input
* Displays top news headlines
* Shows source of each article
* Simple and easy to use

---

## Technologies Used

* Java
* HTTP API (NewsAPI)
* Console-based application

---

## How to Run

1. Compile the program:
   javac NewsFinder.java

2. Run the program:
   java NewsFinder

---

## Setup

1. Go to NewsAPI and create an account
2. Generate your API key
3. Replace the API key in code:

String apiKey = "YOUR_API_KEY";
